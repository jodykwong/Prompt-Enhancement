## 🚀 立即开始

### 安装与配置
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 首次配置
/pe-setup

# 3. 查看帮助
/pe-help
/pe-help-full
```

### 基本使用
```bash
# 执行提示词增强
/pe "你的提示词"

# 使用特定模板
/pe --template fastapi "你的提示词"

# 覆盖标准
/pe --override naming=camelCase "你的提示词"

# 查看检测到的标准
/pe --show-standards
```

### 命令参考
```bash
/pe                 - 基本增强
/pe-setup          - 初始化配置
/pe-help           - 显示帮助
/pe-help-full      - 完整文档
/pe-help TOPIC     - 主题帮助
/pe-quickstart     - 快速指南
/pe-logs           - 查看日志
/pe-clear-cache    - 清除缓存
```

### 常见问题
- Q: 如何重新运行设置？
  A: 运行 `/pe-setup` 或删除 `~/.prompt-enhancement/config.yaml`

- Q: 如何清除缓存？
  A: 运行 `/pe-clear-cache` 或删除 `~/.prompt-enhancement/cache/`

- Q: 如何查看日志？
  A: 运行 `/pe-logs` 或查看 `~/.prompt-enhancement/logs/`
